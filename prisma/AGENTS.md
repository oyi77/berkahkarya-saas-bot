# prisma

## Purpose
Database schema, migrations, and seed data using Prisma ORM with PostgreSQL.

## Key Files

| File | Description |
|------|-------------|
| `schema.prisma` | Full database schema — models, relations, enums |
| `seed.ts` | Database seed script for initial data |

## Subdirectories

| Directory | Purpose |
|-----------|---------|
| `migrations/` | SQL migration files (auto-generated by Prisma) |

## For AI Agents

### Working In This Directory
- After schema changes: `npm run db:generate` then `npm run migrate:dev`
- `npm run db:studio` opens Prisma Studio GUI
- Never edit migration SQL files directly — let Prisma generate them
- `schema.prisma` is the source of truth for all DB models
